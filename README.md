## Description

Nestjs-Pusher is a library that aims to solve a simple problem that is providing a "battery-included" pusher module for Nestjs.
By default, Pusher only gives you a default push mechanism but this library has chunking support which is great for large data amount since the default
payload size is 10 MB.

## Documentation
https://underfisk.github.io/nestjs-pusher/docs/

## Features
- Extensive API
- Event chunking
- Unit tests helper
- Custom decorators and interceptors for REST API

## Installation

```bash
$ npm install nestjs-pusher
$ yarn add nestjs-pusher
```

## Dependency
Since this library uses GraphQl, it should be previously installed in your project as it is considered installed by default.
```bash
npm install --save @nestjs/graphql
```
